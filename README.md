## Installation

To install run

```bash
bash install.sh
```

To Deploy run

```bash
bash start.sh
```

Services with user interfaces will be available at:

- Frontend: <http://localhost:3000>
- Flower: <http://localhost:5555>
- Adminer: <http://localhost:8080>
